# bash-cli-sqlite-script

## Description
A shell script that can CREATE
INSERT and SELECT a sqlite datasource.

## Tech stack
- bash
- sqlite3

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
